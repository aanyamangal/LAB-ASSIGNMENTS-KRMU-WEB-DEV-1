This project is a simple JavaScript quiz game . The quiz runs entirely in the browser console, uses prompt() for input, and displays feedback using alert().

Features

Runs fully in the browser console (no HTML required)

Uses:

Arrays to store questions

Loops to go through each question

Conditionals to check answers

Basic input/output 

toLowerCase() and trim() for clean input

Tracks and displays the final score

Files Included

quiz.js – Main quiz game script

README.md – Documentation and instructions


 How to Run the Quiz


Open any web browser (Chrome, Firefox, Edge, etc.)

Right‑click anywhere on the page and select Inspect

Go to the Console tab

Open the quiz.js file and copy the entire code

Paste it into the console and press Enter

The quiz will start immediately

 How the Quiz Works

Each question is shown using prompt()

Your input is cleaned using:

toLowerCase() – removes case sensitivity

trim() – removes extra spaces

You get instant feedback:

Correct!

Or the correct answer if wrong

At the end, you see your final score out of the total
